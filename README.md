# strongswan-monitoring
Strongswan Monitoring for Telegraf


# Using

Use with Telegraf, it will return you next metrics:

`exec_tunnel_bytes_in`

`exec_tunnel_bytes_out`

`exec_tunnel_packets_in`

`exec_tunnel_packets_out`

`exec_tunnel_status`


| Status        | Value         | 
| ------------- |:-------------:|
| Established   | 0             |
| Installed     | 1             | 
| Another status| 4             |
